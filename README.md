# Chetna Upadhyay Portfolio

A premium, data-driven portfolio for Chetna Upadhyay, a Data Analyst & MBA. 

## Features
- **Modern Design**: Built with React, Tailwind CSS, and Framer Motion.
- **Interactive Sections**: About, Skills, Experience, Projects, and more.
- **Email Integration**: Fully functional contact form using EmailJS.
- **Responsive Layout**: Optimized for all devices.

## Tech Stack
- **Frontend**: React, TanStack Router, Vite
- **Styling**: Vanilla CSS / Tailwind CSS
- **Animations**: Framer Motion
- **Icons**: Lucide React
- **Forms**: Zod, React Hook Form, EmailJS
